Collaborators: @Theojeddy, @Nadzz0, @laila-daher, @YaraWS, @gPrempeh-lab and @esther-frank

### Tech stack

The frontend of this application uses Vite + React + Typescript.

Note: you will need to run the backend too for the app to work.
